## IPFS服务

本地实例化IPFS节点，与ipfs单节点进行交互。

点击上传文件，通过js-http-client将文件直接传进ipfs当中进行存储，返回存储的hash值，通过返回的hash值访问到上传的文件。支持图片上传。
